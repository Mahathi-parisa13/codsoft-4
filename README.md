# 🎬 Movie Recommendation System

This project is a simple AI-based Movie Recommendation System built using Python and Streamlit.

## Features
- Movie recommendations
- Content-based filtering
- Simple UI using Streamlit

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
